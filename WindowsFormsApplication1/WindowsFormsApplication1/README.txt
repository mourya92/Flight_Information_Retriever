This is a windows desktop application developed using C# and Visual Studio 2010 IDE. 

This application is used to parse a scrambled input file,"sample.txt", to obtain certain information and print it in a readable format. 

This application takes a line number as input and parses that particular line in input file. 

Basically, input file contains information about several flights in each line.

This application retrieves Flight Number, Carrier, Departure, Arrival places and timings etc.,

GUI is created using drag and drop. 

Parsing program is in Airport.cs.

To run: 
	- Add the files Airport.cs, Form1.cs program.cs to an empty C# project using Visual Studio
	- Run the project.
